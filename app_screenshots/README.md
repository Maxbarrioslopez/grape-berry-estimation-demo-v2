# Capturas de la app Android

Set actualizado de pantallas capturado desde el emulador `Pixel_7` con la APK `debug` en `DEMO_MODE`.

La app no requiere usuario ni contrasena en este modo. Al abrirla aparece el acuerdo academico obligatorio; se marco la casilla de aceptacion y luego se presiono `Continue` para entrar al flujo.

## Notas

- Las capturas antiguas fueron reemplazadas por nombres claros y secuenciales.
- No se dejo visible el teclado en las pantallas finales del flujo principal.
- Para el proceso de imagen se cargaron fotos de prueba en la galeria del emulador y se recargo el escaneo multimedia antes de seleccionarlas.
- El resultado mostrado corresponde a una corrida local en modo demo; no se uso backend ni sincronizacion remota.
